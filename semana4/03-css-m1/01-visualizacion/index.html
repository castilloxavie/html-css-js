Visualización de los Elementos

Display

La propiedad display define el tipo de caja que se va a utilizar para un elemento.

Visibility

La propiedad visibility especifica si un elemento de caja es visible.

Diferencia entre display y visibility

La propiedad display define el tipo de caja que se va a utilizar para un elemento, mientras que la propiedad visibility especifica si un elemento de caja es visible.

Opacity

La propiedad opacity especifica la opacidad de un elemento.
